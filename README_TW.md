# 捷運大富翁

## 介紹

由IZCC共同開發，作為社團內具特色的一個活動，在社團內的春遊、暑訓、秋遊、寒訓、茶會等活動皆可使用，遊戲結合了現實的團隊活動與資訊科技的應用，目標是希望能讓參與者藉由資訊社的活動體驗資訊科技帶來的娛樂與應用。
